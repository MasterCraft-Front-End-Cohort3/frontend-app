# Frontend App

## Project Setup Instructions

1. Clone the repo:
   ```sh
   git clone https://github.com/KimemiaT/frontend-app.git
   cd frontend-app
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the app:
   ```sh
   npm start
   ```
4. Run tests:
   ```sh
   npm test
   ```
5. Lint and format:
   ```sh
   npm run lint
   npm run format
   ```
